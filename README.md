## Broberg Consult Limited Web Application

## Overview
The Broberg Consult Ltd. web app is a professional platform showcasing the company's engineering consultancy services, projects, and expertise.

## Features

- Responsive Design: Optimized for various devices, including desktops, laptops, tablets, and smartphones.
- Home Page Section: Prominent hero section on the homepage to showcase the company's tagline and services.
- Projects Section: Showcase of notable projects, including images and descriptions.
- About Us Section: Overview of the company's history, mission, and values.
- Contact Form: Simple contact form for visitors to send messages, and also a display of google map to the company's location.

## Additional Features

- Social Media Links: Links to the company's social media profiles.

## Technical Features

- Contact Form Validation: Validation to ensure valid information entry.
- Responsive Images: Images that resize and adapt to different screen sizes and devices.

## Getting Started

1. Clone the repository: git clone https://github.com/your-repo/broberg-consult-ltd.git
2. Install dependencies: npm install
3. Start the development server: npm start
